# eSIMDZ — Pro Landing (AR/EN/FR)

- تبديل لغات فوري (AR/EN/FR) عبر `i18n.json`.
- روابطك الاجتماعية مضمنة (Instagram/Telegram/TikTok/Facebook).
- واتساب للشراء الفوري برسائل ديناميكية حسب اللغة.
- ملفات SEO: `robots.txt` و `sitemap.xml` + OG image.

## إطلاق على GitHub Pages
1) أنشئ مستودعًا عامًّا (مثلاً `esimdz-pro`).
2) ارفع **كل محتويات** هذا المجلد (لا ترفع المجلد الأب).
3) Settings → Pages → Deploy from a branch → فرع `main` ومسار `/root`.
4) استبدل في `index.html` معرف GA4 (`G-XXXXXXX`) وPixel (`1234567890`).
